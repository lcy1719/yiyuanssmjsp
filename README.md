### 送万字文档+医院系统+SSM+JSP
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL
技术栈：Java、MySQL、Spring、SpringMVC、Mybatis/MybatisPlus、JSP、JavaScript、CSS、文件上传
总共18张数据库表

系统功能模块：
本系统包含管理员、护士、医生、药房4个角色。
管理员角色：
1. 用户管理：管理系统中所有用户的信息，包括添加、删除和修改用户。
2. 药房管理：管理药房信息，包括添加、删除和修改药房信息。
3. 护士管理：管理护士信息，包括添加、删除和修改护士信息。
4. 医生管理：管理医生信息，包括添加、删除和修改医生信息。
5. 病人信息管理：管理病人信息，包括添加、删除和修改病人信息。
6. 科室信息管理：管理科室信息，包括添加、删除和修改科室信息。
7. 挂号管理：管理挂号信息，包括添加、删除和修改挂号信息。
8. 诊断信息管理：管理诊断信息，包括添加、删除和修改诊断信息。
9. 病例库管理：管理病例库信息，包括添加、删除和修改病例库信息。
10. 开药信息管理：管理开药信息，包括添加、删除和修改开药信息。
11. 药品信息管理：管理药品信息，包括添加、删除和修改药品信息。
12. 收费信息管理：管理收费信息，包括添加、删除和修改收费信息。
护士角色：
1. 病人信息管理：管理病人信息，包括查看、修改和删除病人信息。
2. 挂号管理：管理挂号信息，包括查看、修改和删除挂号信息。
3. 个人信息管理：管理自己的个人信息，包括查看、修改和删除个人信息。
医生角色：
1. 挂号管理：查看挂号信息。
2. 诊断信息管理：管理诊断信息，包括修改和删除诊断信息。
3. 病例库管理：管理病例库信息，包括查看、修改和删除病例库信息。
4. 开药信息管理：管理开药信息，包括查看、修改和删除开药信息。
药房角色：
1. 挂号管理：查看挂号信息。
2. 药品信息管理：管理药品信息，包括查看、修改和删除药品信息。
3. 收费信息管理：管理收费信息，包括查看、修改和删除收费信息。
